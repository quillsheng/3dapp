Fat-Free Framework for PHP

The f3 framework is equipped with a fast and easy to use template engine. f3's mods work with f3's data mapping and sql tools to do more sophisticated communication with a variety of database engines. Source: https://fatfreeframework.com/3.7/home

index.php is the entry file for the project, referencing the f3 MVC framework and loading some system configuration such as: routing configuration;

  • The app directory is the core of the system, the controls are defined in controllers, the models are defined in models and the templates are defined in views. The database connection is in database.php in helpers.
  • The system uses a SQLite database, and some of the data on the page is read from and displayed in SQLite.
  • static resources directory, this directory is the entire system resources directory, such as js, css, and image, etc. are placed here
  • tmp is created automatically when the system is running. The main purpose here is for f3 to convert the corresponding pages in views into the corresponding templates and save them in this directory, so that the next time it is run, the pages can be rendered quickly.
